# 🌦️ Real-Time Weather Data Extraction Project

## 📌 Project Overview
This project demonstrates real-time data extraction using a weather API and visualization of trends.

The objective was to:

- Extract real-time data using API  
- Clean and process the data  
- Visualize temperature and humidity trends  
- Understand real-world applications  

---

## 🔍 Analysis Performed

### Data Extraction
- Used OpenWeatherMap API  
- Retrieved temperature and humidity data  

### Data Processing
- Cleaned dataset  
- Removed duplicates  
- Simulated variations for trend analysis  

### Visualization
- Temperature trend line chart  
- Humidity bar chart  

---

## 📊 Key Insights
- Weather data shows slight variation over time  
- Temperature and humidity trends can be visualized  
- Real-time data enables dynamic analysis  
- API-based automation simplifies data collection  
- Useful for decision-making systems
